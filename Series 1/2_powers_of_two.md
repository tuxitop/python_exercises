‮برنامه‌ای بنویسید که عدد n را از ورودی بخواند و اولین توان عدد دو را که از n بزرگتر است چاپ کند.

### ورودی

‮در تنها خط ورودی عدد n آمده است.

$$
1 \leq n \leq 10^9
$$

### خروجی

در خروجی جواب خواسته شده را چاپ کنید.

## مثال

### ورودی نمونه ۱

```
95
```

### خروجی نمونه ۱

```
128
```

### ورودی نمونه ۲

```
1010
```

### خروجی نمونه ۲

```
1024
```
